# SEO Dashboard

This repository serves as an SEO dashboard for tracking key metrics, analyzing keywords, and planning content strategy. It contains reports, tools, and templates for daily SEO manager work.

## Structure

- **reports/** - Monthly reports, keyword analysis, audit results
- **tools/** - Excel file templates and checklists for website audits, backlink checks
- **content-strategy/** - Blog ideas, content calendar, content optimization notes
- **assets/charts/** - Graphs and visualizations from reports

## Usage

1. Browse individual sections according to your needs
2. In the `tools/` folder you'll find templates for daily work
3. In `reports/` are stored results of previous analyses
4. In `content-strategy/` plan your content strategy
